# caplifi-approve — Grok Build plugin

[Caplifi Approve](https://caplifi.com/approve/) — open contract **headgate.approve.v1**.

## One rule

Agents propose freely. Nothing irreversible runs because a model decided it should.  
It runs when a human-authorized gate allows it.

## Skills

| Skill | When |
|-------|------|
| `caplifi-approve` | Propose access/execute, format envelopes, honor deny/approve |

## Spec

https://caplifi.com/approve/spec/

## Install

```bash
grok plugin install ./plugins/caplifi-approve --trust
```

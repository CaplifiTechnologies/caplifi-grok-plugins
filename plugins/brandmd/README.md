# brandmd — Grok Build plugin

Open [BRAND.md](https://caplifi.com/build/brandmd/) standard for machine-readable brand identity.

## Skills

| Skill | When |
|-------|------|
| `brandmd-apply` | Writing or designing with a brand repo present |
| `brandmd-audit` | Checking BRAND.md conformance / completeness |

## Commands

- `/brandmd` — quick pointer to apply + audit workflow

## Spec

https://caplifi.com/build/brandmd/ · SPEC on the public standard site

## Install

```bash
grok plugin install ./plugins/brandmd --trust
```

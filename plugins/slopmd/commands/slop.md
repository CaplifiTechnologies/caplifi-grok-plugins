# /slop

SLOP + INSTIGATE/HANDOFF helper.

1. Ask: execute now (**INSTIGATE** / `type: instigate`) or brief only (**HANDOFF** / `type: handoff`)?  
2. Emit a valid envelope or doc with correct first token.  
3. Never write irreversible work into an ungated lane.

# slopmd — Grok Build plugin

[SLOP.md](https://eclecticventures.net/slopmd/) — transport-agnostic agent handoff envelopes.  
Plus the **INSTIGATE / HANDOFF** document contract (execute vs discuss).

## Invariant

**Fuzzy layers inform. Deterministic code gates anything irreversible.**

## Skills

| Skill | When |
|-------|------|
| `slop-envelope` | Author or parse SLOP envelopes |
| `instigate-handoff` | Write INSTIGATE vs HANDOFF docs correctly |

## Install

```bash
grok plugin install ./plugins/slopmd --trust
```
